linting and fixing code for all languages

install coala

simply install coala

If any kind of dependency errors occur, use a virtualenv!

For the full installation documentation, check coala.io/install 

$ pip3 install coala-bears

You can append --pre to that command to get the latest prerelease straight from master!

Consider using a virtualenv:

$ pip3 install virtualenv
$ virtualenv venv        # On Linux
$ venv\scripts\activate  # On Windows
get it in docker

The most simple way to get the full functionality if you use docker.

$ docker run -ti -v $(pwd):/app --workdir=/app coala/base coala --version

For more information, check out our docker documentation .
